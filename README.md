# KF Öresund — Invoice Automation Tool

A desktop tool built for **KF Öresund** to automate invoice handling and reduce manual workload in their day-to-day operations.

![Java](https://img.shields.io/badge/Java-007396?logo=openjdk&logoColor=white)
![JavaFX](https://img.shields.io/badge/JavaFX-FF0000?logo=java&logoColor=white)

---

## What it does

Automates the parts of invoice handling that were previously done by hand — parsing inputs, generating outputs in the required format, and reducing the time per invoice from minutes to seconds.

## Skills demonstrated

- End-to-end Java application development
- Real-world problem solving for an actual stakeholder
- Desktop UI development
- File and data parsing

## Tech stack

- **Language:** Java
- **UI:** JavaFX
- **Use case:** Internal automation tool
